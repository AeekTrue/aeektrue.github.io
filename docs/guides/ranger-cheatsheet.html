General

ShortcutDescription

QQuit Ranger

Movement

ShortcutDescription
kup
jdown
hparent directory
lsubdirectory

Ggo t bottom of list
Jhalf page down
Khalf page up
HHistory Back
LHistory Forward
~Switch the view

File Operations

ShortcutDescription

Commands

ShortcutDescription
:Execute Range Command
!Execute Shell Command
chmodChange file Permissions
duDisk Usage Current Directory
SRun the terminal in your current ranger window (exit to go back to ranger)

Tabs

ShortcutDescription
C-nCreate new tab
C-wClose current tab
tabNext tab
shift + tabPrevious tab
alt + [n]goto / create [n] tab

File substituting

ShortcutDescription
%fSubstitute highlighted file
%dSubstitute current directory
%sSubstitute currently selected files
%tSubstitute currently tagged files

Example for substitution

:bulkrename %s

ShortcutDescription
